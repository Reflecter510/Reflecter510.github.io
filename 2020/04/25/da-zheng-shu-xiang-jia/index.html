大整数相加

模拟
algorithm
发布日期:   2020-04-25
更新日期:   2020-04-25
文章字数:   684
阅读时长:   3 分

问题描述

将两个字符串形式的大整数相加，输出结果。

示例
输入

2435354365
13243

输出

2435367608

模拟

按照竖式加法运算进行模拟，用一个长度为 max(a.length,b.length)+1 的字符数组ans保存结果。

然后从两个字符串的末尾逐位进行运算，首先处理两个字符串重合的部分（即i和j都大于等于0），取当前位的相加结果 mod 10 ，然后相加结果 / 10作为进位。

然后就是将a或b中剩下的数字加入ans中。

完了之后就剩最高位是否进位，是的话就继续存个1。

最后是调整数组的大小，最高位有无进位会影响ans第一个元素是否存在，可能需要对数组进行移动以消除空白字符。

另一种做法是：先将两个字符串的长度对齐，具体为先将结果数组（长度为max(a.length,b.length)）用'0'填充，然后放入长度较小的数字字符串，之后便是与另一个数字字符串进行相加操作，最后根据最高位是否进位来判断是否在数组首部插入进位1。

package day4;

import java.util.Scanner;

public class BigIntAdd {

    static char[] addBigInt(char[] a, char[] b)
    {
        if(a==null)return b;
        if(b==null)return a;
        int beyond = 0;
        int i = a.length-1;
        int j = b.length-1;
        char[] ans = new char[i>j ? i+2: j+2];
        int k = ans.length-1;
        while(i>=0 && j>=0)                   // a和b共同部分
        {
            int num = (a[i--] - '0') + (b[j--] - '0') + beyond;
            ans[k--] = (char)(num%10+'0');
            beyond = num/10;
        }
        while(i>=0) {                         // 若 a 最长
            int num = (a[i--] - '0') + beyond;
            ans[k--] = (char)(num%10+'0');
            beyond = num/10;
        }
        while(j>=0) {                         // 若 b 最长
            int num = (b[j--] - '0') + beyond;
            ans[k--] = (char)(num%10+'0');
            beyond = num/10;
        }
        if(beyond>0)                          // 最高位进位
            ans[k--] = (char)(beyond+'0');
        if(k==0)                              // 若最高位没有进位
        {
        	char tt[]=new char[ans.length-1];
        	System.arraycopy(ans, 1, tt, 0, ans.length-1);
        	return tt;
        }
        return ans;
    }
    /*
     * a="2435354365";
     * b="13243";
     * result=000013243
     */
    
    static  char[] addBigInt2(char a[],char b[])
    {
        if(a==null) return b;
        if(b==null) return a;
        char[] temp=null;
        int len=a.length>b.length? a.length:b.length;
        char result[]=new char[len];
        for(int i=0;i<len;i++)
            result[i]='0';
        if(a.length>b.length)
        {
            int k=len-1;
            for(int i=b.length-1;i>=0;i--)
                result[k--]=b[i];
            temp=a;
        }
        else
        {
            int k=len-1;
            for(int i=a.length-1;i>=0;i--)
                result[k--]=a[i];
            temp=b;
        }
        int c=0;//  c 是进位
        for(int i=temp.length-1;i>=0;i--)
        {
            int  t=(temp[i]-'0')+(result[i]-'0')+c;
            if(t<10)  {  result[i]=(char)(t+'0'); c=0;}
            else
            {
                c=1;
                result[i]=(char)(t-10+'0');
            }
            
        }
        if(c==1)
        {
        	char tt[]=new char[len+1];
        	System.arraycopy(result, 0, tt, 1, len);
        	tt[0]='1';
        	return tt;
        }
        
        return result;
    }
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Scanner sc = new Scanner(System.in);
        
        String a = sc.nextLine();
        String b = sc.nextLine();
        
        char[] ans = addBigInt(a.toCharArray(),b.toCharArray());
        System.out.println(ans);
    }
}
