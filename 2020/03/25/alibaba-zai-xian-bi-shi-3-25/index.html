题目一 矩阵列差最小值

小强有一个 3 x n 的矩阵 a，他将 a 中每列的三个数字中取出一个按顺序组成一个长度为 n 的数组 b，即$b_i$可以是$a_{1,i} 、 a_{2,i} 、 a_{3,i}$其中任意一个。问$\sum_{i=0}^{n-1}|b_i-b_{i+1}|$的最小值是多少？

输入描述

第一行，一个正整数n。
第二行到第四行输入一个 3 x n 的矩阵 a，每行输入 n个正整数。
$2<=n<=10^5, 1<=a_{i,j}<=10^9$

输出描述

一行一个正整数表示答案

示例一
输入

5
5 9 5 4 4
4 7 4 10 3
2 10 9 2 3

输出

5

说明

数组 b 可以为[5,7,5,4,4]，答案是5.

动态规划

用回溯只过了30%的样例，这也是最直接的可以想到的方法。事后听说这题是用动态规划的，当时想的是贪心是肯定不能贪，把示例里的一三列调换一下就知道了，但是动态规划的话确实没思路，找不到最优子结构是什么，$n$等于1的时候答案是0，$n$等于2的时候是取两列中数字相差最小的，$n$等于3的时候就……这时候$n$等于3的解可能并不包括$n$等于2的时候，因为$n$等于2的时候又相当于是贪心了一次。

我曾经想最简的子问题会不会不是一列，看了题解，是用二维数组来保存 $dp$ 的状态，最小的子问题确实只有一列。

为什么要用二维数组来保存状态呢？
如果只看示例的前三列，可以发现除了$[5,7,5]$是最小的4，还有$[5,9,9]$也是最小的4，更进一步，如果前三列这个子问题扩展到前四列，可以发现这俩个子问题之间的关联就是前三列子问题的最后一列，取5或者取9或4会关系到后面的一列的取值。因此，共三行，取3种状态。

还有就是问题的规模，也需要一维来记录，所以得到 $dp$ 数组的格式：$dp[i][j]$，其中 i 表示第 $i$ 行；$j$ 则表示问题的规模 $1…n-1$。

即$dp[i][j]$表示选择第 $i$ 行第 $j$ 列作为当前问题（规模为$j$）的解是能得到的最小值；

于是有状态转移方程：

$dp[i][j] = min(dp[k][j-1]+abs(nums[i][j]-nums[k][j-1])); k\in{0,1,2}$

  • $j=1..n-1$
  • $i,k=0..2$

其中，每列中的每一个取值视为一种状态。对于当前问题的每一种状态，都可能由规模减一的子问题的三种状态得来，每种状态加上原数组两状态的差值，取三个和中最小的即为当前问题当前状态的最优解。

  • 当前问题的每一种状态：$nums[i][j]$
  • 规模减一的子问题：$dp[k][j-1]$，有三种状态$k=0,1,2$
  • 当前问题当前状态：$dp[i][j] $
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

ll nums[3][100001], dp[3][100001];
ll ans = LLONG_MAX;
int n;

int main()
{
    cin >> n;
    for (int i = 0; i < 3; i++)
        for (int j = 0; j < n; j++)
        {
            scanf("%d", &nums[i][j]);
        }

    dp[0][0] = dp[1][0] = dp[2][0] = 0; // 第一列初始化为0，最小的子问题
    
    for (int j = 1; j < n; j++)
    {
        for (int i = 0; i < 3; i++)
        {
            dp[i][j] = INT_MAX;
            for (int k = 0; k < 3; k++)
                dp[i][j] = min(dp[i][j], dp[k][j - 1] + abs(nums[i][j] - nums[k][j - 1]));
        }
    }
    
    for (int i = 0; i < 3; i++)
    {
        ans = min(ans, dp[i][n - 1]);
    }
    cout << ans << endl;
    return 0;
}

题目二 隐藏的等差数列矩阵

给出一个二维矩阵，这个矩阵的每一行和每一列都是一个独立的等差数列，但是其中的一些数字被隐藏住了，现在需要你进行推理将那些被隐藏但是可以被唯一确定的数字填上，然后来回答对某个位置上数字的询问。

输入描述

第一行，n,m,q分别表示矩阵的行数，列数和查询的条数。
接下来的n行，每行m个数表示这个矩阵，0表示缺失数据。
接下来q行，每行两个数字i,j表示对矩阵中第i行第j列的数字进行查询。

输出描述

如果可以确定该位置的数字，则输出数字，如果不能确定则输出Unknown。

示例一
输入

2 3 6
1 0 3
0 0 0
1 1
1 2
1 3
2 1
2 2
2 3

输出

1
2
3
Unknown
Unknown
Unknown

None

下面这个暴力解法，只要每行有两个数就能复原这一行，每列有两个数就能复原这一列。估摸着也许能过30%的样例。

#include<bits/stdc++.h>
using namespace std;

int nums[502][502];

int main()
{
    int n,m,q;
    cin>>n>>m>>q;

    for(int i=0;i<n;i++){
        // 按行复原
        int left=-1,lval=0,right=-1,rval=0;
        for(int j=0;j<m;j++){
            cin >> nums[i][j];
            if(nums[i][j]){
                if(left == -1){
                    left = j;
                    lval = nums[i][j];
                }
                else if(right == -1){
                    right = j;
                    rval = nums[i][j];
                }
            } 
        }
        
        if(left!=-1 && right!=-1){
            int d = (rval-lval)/(right-left);
            for(int j=left-1;j>=0;j--){   // 0..left
                nums[i][j] = nums[i][j+1]-d;
            }
            for(int j=left+1;j<right;j++){  // left..right
                nums[i][j] = nums[i][j-1]+d;
            }
            for(int j=right+1;j<m;j++){  // right..m
                nums[i][j] = nums[i][j-1]+d;
            }
        }
    }

    //  按列复原 
    for(int j=0;j<m;j++){
        int left=-1,lval=0,right=-1,rval=0;
        for(int i=0;i<n;i++){
            if(nums[i][j]){
                if(left == -1){
                    left = i;
                    lval = nums[i][j];
                }
                else if(right == -1){
                    right = i;
                    rval = nums[i][j];
                }
            } 
        }
        if(left!=-1 && right!=-1){
            int d = (rval-lval)/(right-left);
            for(int i=left-1;i>=0;i--){   // 0..left
                nums[i][j] = nums[i+1][j]-d;
            }
            for(int i=left+1;i<right;i++){  // left..right
                nums[i][j] = nums[i-1][j]+d;
            }
            for(int i=right+1;i<n;i++){  // right..n
                nums[i][j] = nums[i-1][j]+d;
            }
        }
    }

    while(q--){
        int x,y;
        cin>>x>>y;
        x--; y--;
        if(nums[x][y])cout<<nums[x][y]<<endl;
        else
        {
            cout<<"Unknown"<<endl;
        }
    }

    return 0;
}
