题目一 扑克牌

今天小强从一副扑克牌里拿出一叠，其中包括A,2,3,…,10，各四张，其中A代表1。他从这一叠中抽出一些牌给小明，并告诉小明每次可以按照下列方式打出一些牌：

  • 单牌：一张牌，例如3.
  • 对子：数字相同的两张牌，例如77
  • 顺子：数字连续的五张牌，例如A2345.

现在小强想知道最少打出多少次牌可以打光手里的牌。

输入描述

一行十个空格分隔的整数$A_1$,$A_2$,$A_3$,…,$A_{10}$，分别代表为A,2,3,…,10的个数。
$0<=A_1,A_2,…，A_{10}<=4$
保证手里至少有一张牌

输出描述

仅一行表示答案。

示例1
输入

1 1 1 2 2 2 2 2 1 1

输出

3

说明

打出了三个顺子，分别为：A2345,45678,678910。

DFS

写了一个贪心DFS来搜，没有 OJ 可以测，先记录一下。

如果能打5张牌就打，其次2张，最后1张。

#include <bits/stdc++.h>
using namespace std;

vector<int> cnt(10, 0);
int ans = INT_MAX;

bool check(int cur)
{
    for (int i = cur; i < cur + 5; i++)
    {
        if (0 == cnt[i])
            return false;
    }
    return true;
}

void dfs(int sum, int cur)
{
    while (cnt[cur] == 0)
        cur++; // 0张牌的不计

    if (cur >= 10) // 递归出口，所有牌都打完
    {
        ans = min(ans, sum); // 取最小
        return ;
    }

    if (cur + 4 < 10 && check(cur)) // 连续打5张
    {
        int temp = cur;
        for (int i = temp; i <= temp + 4; i++)
        {
            cnt[i]--;
            if (cnt[i] == 0)
                cur++;
        }
        dfs(sum + 1, cur);
    }
    else if (cnt[cur] >= 2) //打两张一样的
    {
        cnt[cur] -= 2;
        if (cnt[cur] == 0)
            cur++;
        dfs(sum + 1, cur);
    }
    else if (cnt[cur] > 0) //打一张
    {
        cnt[cur]--;
        if (cnt[cur] == 0)
            cur++;
        dfs(sum + 1, cur);
    }
}

int main()
{
    for (int i = 0; i < 10; i++)
    {
        cin >> cnt[i];
    }
    dfs(0, 0);
    cout << ans << endl;
    return 0;
}

题目二 最长旋律

小强喜欢弹钢琴，一段旋律中的每个音符都可以用一个小写英文字母来表示。当组成一段旋律的字符ASCII码是非递减的，旋律被称为高昂的，例如aaa,bcd。

现在小强已经学会了n段高昂的旋律，他想利用它们拼接出一个尽可能长的高昂旋律，问最长长度是多少？

输入描述

第一行一个正整数n，含义如题目所述。
接下来n行每行一个字符串，保证每个字符串的字符的ASCII码是非递减的。
$1<=n<=10^6$
保证所有字符串的长度之和不超过$10^6$且仅由小写字母构成。

输出描述

仅一行一个整数代表答案

示例1
输入

4
aaa
bcd
zzz
bcdef

输出

11

说明

将1,4,3段字符串拼接到一起，长度为11。
