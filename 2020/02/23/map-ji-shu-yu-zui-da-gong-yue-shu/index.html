map计数与最大公约数

leetcode 数学 Map
algorithm
发布日期:   2020-02-23
更新日期:   2020-03-11
文章字数:   424
阅读时长:   1 分

题目（Leetcode 914. 卡牌分组）

给定一副牌，每张牌上都写着一个整数。

此时，你需要选定一个数字 X，使我们可以将整副牌按下述规则分成 1 组或更多组：

  • 每组都有 X 张牌。
  • 组内所有的牌上都写着相同的整数。
    仅当你可选的 X >= 2 时返回 true。

示例 1：

输入：[1,2,3,4,4,3,2,1]
输出：true
解释：可行的分组是 [1,1]，[2,2]，[3,3]，[4,4]

示例 2：

输入：[1,1,1,2,2,2,3,3]
输出：false
解释：没有满足要求的分组。

示例 3：

输入：[1]
输出：false
解释：没有满足要求的分组。

示例 4：

输入：[1,1]
输出：true
解释：可行的分组是 [1,1]

示例 5：

输入：[1,1,2,2,2,2]
输出：true
解释：可行的分组是 [1,1]，[2,2]，[2,2]

提示：

  1. 1 <= deck.length <= 10000
  2. 0 <= deck[i] < 10000

解答

考察读题意的一道题，统计每种数字的个数，并求它们的最大公约数，若不为1则true。

class Solution {
public:
    int gcd(int x,int y){
        return x==0?y:gcd(y%x,x);
    }
    bool hasGroupsSizeX(vector<int>& deck) {
        map<int,int> cnt;
        for(int i=0;i<deck.size();i++)cnt[deck[i]]++;

        if(cnt.size()==deck.size())return false;
        map<int,int>::iterator it=cnt.begin();
        int atom=it->second;

        it++;
        for(;it!=cnt.end();it++){
            //if(it->second%atom!=0)return false;  //不能这么判断，要最大公约数
            //错误判断1：每种数字的个数相等
            //错误判断2：每种数字的个数是最小个数的整数倍
            //最终：求所有个数的最大公约数，若存在则true
            atom=gcd(atom,it->second);
        }
        if(atom!=1)
            return true;
        else 
            return false;
    }
};
